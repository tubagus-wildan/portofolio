Gelati Ilmu Mobile

Video Game ini supoort Android Kit-Kat - Android 12

Video Game ini dibuat untuk melihat efektifitas video game terhadap pelajaran khususnya bahasa jawa banten kelas 4(empat) sekolah dasar

Penjelasan

Video game ini dibangun dengan menggunakan engine unity, dengan menggunakan sudut pandang orang pertama yang dapat membuat permainan lebih asik. Video game ini dibuat pertamakali pada tahun 2022 dengan tujuan penelitian tentang efektifitas video game terhadap pendidikan dengan khususnya mata pelajaran bahasa jawa banten, permainan ini di targetkan untuk anak-anak dengan umur 6-11 tahun

Spesifikasi

Minimum
Sistem Oprasi Android 4.4.4 (KitKat).
Chipset Qualcomm SDM450 Snapdragon 450 atau MediaTek Helio P35.
CPU / Prosesor Octa-core 1.8 GHz Cortex-A53
GPU / Prosesor Grafis Adreno 506.
RAM 2 GB.
Memory bebas 300 MB.
Rekomendasi
Sistem Oprasi Android 11 (Red Velvet Cake) atau diatas nya.
Chipset Qualcomm Snapdragon 665 atau Unisoc Tiger T616
CPU / Prosesor Octa-core 2x2.0 GHz Cortex A75 atau di atasnya
GPU / Prosesor Grafis Adreno 618.
RAM 4 GB atau di atasnya.
Memory bebas 1 GB.
